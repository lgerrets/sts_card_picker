## Install

> Install-Module -Name Set-PsEnv

> Set-Alias -Name dotenv -Value Set-PsEnv

> python -m venv env

> ./env/scripts/activate

> pip install -r requirements.txt

## Run

> dotenv

> ./env/scripts/activate

> python sts_ml/deck_history.py

> python sts_ml/train.py

> python sts_ml/infer.py
